Sta je HTML?

HTML je skraćenica koja predstavlja akronim pojma HyperText Markup Language, što je osnovni, opisni jezik za kreiranje HTML dokumenata. Iz samog naziva ovog pojma može se zaključiti mnogo o HTML jeziku.

HyperText Markup Language, sačinjen je iz dva pojma:

  • HyperText
  • Markup Language
